Then, we want to generalise the moment estimation to statistics estimation, in this case, we can choose `Indirect Inference` method to estimate the parameters. 2. Indirect Inference Method

First, given an initial set of parameters θ∈Θ⊂ℝd we simulate a series of data {yi(θ)}i=1,2,...,N following the model F(θ).

Then, we want to find an auxiliary model G(μ), where μ∈U⊂ℝr where r≥d, so that a direct method of estimation is feasible, supposing the target function to be maximised is Q(μ) then:

μ̃ G=argmaxμ∈ℝr  Q(y;μ)

For the real data set {yt}t=1,2,...,T, we can easily compute μ̃ G:

μ̃ G=argmaxμ∈ℝr  Q({yt}t=1,2,...T;μ)

The same for the simulated pseudo data:

μ̃ G(θ)=argmaxμ∈ℝr  Q({yi(θ)}i=1,2,...N;μ)

Our problem becomes:

θ̂ =argminθ∈ℝd[μ̃ G(θ)−μ̃ G]TΩ[μ̃ G(θ)−μ̃ G]

where Ω is the symmetric positive definite weight matrix.
